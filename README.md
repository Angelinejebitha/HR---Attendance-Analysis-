# Employee Attendance Analysis

**📊 Project Overview**

This project analyzes employee attendance data to identify patterns and trends.
 
**🎯 Objective**
- Monitor and evaluate key workforce metrics including attendance rate, WFH share, and sick leave rate
- Analyze attendance trends and employee work patterns to identify behavioral insights
- Detect absenteeism trends and potential productivity gaps
- Support HR decision-making through data-driven insights and workforce analytics
- Prepare structured MIS report and KPI summaries to support operational monitoring and business reporting.

**💻⚙️ Tools & Processes**
- Power BI
- DAX
- Power Query
- Excel
- Data Cleaning
- Data Preparation
- Data Validation
- Data Visualization
- KPI Analysis
- Dashboard Reporting
- MIS Reporting
- Workforce Reporting

**📌 Key Insights**
- Attendance remained stable above 89% throughout the 3-month period
- WFH participation peaked during May while remaining relatively stable overall
- Attendance rate declined during May before showing moderate recovery in June
- Employee-level attendance analysis identified a few employees with comparatively lower attendance percentages
- Fridays show relatively higher WFH participation compared to other weekdays


**MIS Reporting**

This project also includes structured MIS reporting using Excel, covering KPI analysis, attendance trend analysis, workforce reporting, insight generation, and business recommendations
